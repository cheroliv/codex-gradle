== Livrables

=== 1. ConvertToMarkdownTask.kt (173 lignes)

Conversion ligne à ligne avec préservation :
- Hiérarchie AsciiDoc `=`/`==`/`===` → Markdown `#`/`##`/`###`
- Blocs de code `[source,java]` + `----` → ```java ... ```
- Tableaux `|===` / `|` → Markdown tables `| ... |`
- Images `image:alt[path]` → `![alt](path)`
- Liens `link:` / `http` → `[text](url)`
- Admonitions `[NOTE]` / `[WARNING]` / `[TIP]` → `> **Note**` blockquote
- Listes `* item` → `- item`
- Inline formatting : **bold**, *italic*, `code`, _underline_
