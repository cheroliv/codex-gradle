== Objectif

Détection automatique de la zone (OSS vs CSS) par chemin physique
et routage des licences dans le pipeline.

== Livrables

=== 1. LicenseZoneDetector.kt (20 lignes)

- Enum `LicenseZone` : OSS, CSS, UNKNOWN
- Détection par chemin : `/foundry/OSS/` → OSS, `/foundry/CSS/` → CSS
- Mapping licence : OSS → `Apache-2.0`, CSS → `PROPRIETARY`

=== 2. CodexExtension.kt

Extension Gradle `codex { zone = ... }` avec `Property<LicenseZone>`.
Accessible par les builds consommateurs du plugin.

=== 3. CodexPlugin.kt mis à jour

- Détection de zone au chargement (`project.projectDir.absolutePath`)
- Création de l'extension `codex`
- Injection `licenseName` dans `chunkDocument`

=== 4. ChunkDocumentTask.kt mis à jour

- Nouvelle propriété `licenseName: @Input Property<String>`
- Propagation automatique : `DocumentChunk.license = licenseName.get()`
