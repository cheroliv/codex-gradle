multi-format (JSON-L, Markdown, AsciiDoc).

== Livrables

=== 1. ExportKnowledgeBaseTask.kt (215 lignes)

Agrégation chunks → 3 formats cohérents :

*JSON-L* :
- `@context` schema.org/codex avec `isPartOf` mapping
- `@graph` avec `@id` URN (`urn:codex:chk-xxxxxxxx`)
- `@type: DocumentChunk`, champs : sectionPath, headingLevel, text, codeBlocks, entities, license

*Markdown* :
- Hiérarchie `#`/`##`/`###` reconstruite depuis sectionPath
- Code blocks en ```text ... ``` 
- Overlap en `> _Suite : ..._` blockquote
- Séparateurs `---` entre chapitres (headingLevel 1)

*AsciiDoc* :
- `=`/`==`/`===` hiérarchie
- `[source,text]` + `----` pour code blocks
- `[TIP]` pour overlap
- Séparateurs `'''` entre sections

Tri sémantique : sectionSortKey avec numéros paddés (1.2 → 0001.0002)
