== Objectif

Détection de la hiérarchie des titres dans `ExtractBookStructureTask` via analyse
des tailles et graisses de police dans le flux PDF (PDFBox `writeString` + `TextPosition`).

== Livrables

=== 1. Fix split-brain buildSrc/ ↔ doc-pipeline/

* `AsciiDocToJsonLddTask` et `ImportBookSqlTask` migrées de `buildSrc/` → `doc-pipeline/`
* `DocNode` extrait dans `DocNode.kt` (top-level shared)
* Duplicats `ExtractTextTask` et `ExtractBookStructureTask` supprimés de `buildSrc/`
* Convention plugin `buildSrc/` nettoyé (plus d'enregistrement direct de tâches)
* `CodexPlugin` enregistre maintenant les 4 tâches : `extractText`, `extractBookStructure`, `asciiDocToJsonLdd`, `importBookSql`

=== 2. FontStyleDetector

* Nouveau fichier `FontStyle.kt` : enum `FontStyle` (NORMAL, BOLD, ITALIC, BOLD_ITALIC, MONOSPACE) + `FontStyleDetector`
* Détection via fontName (lowercase) :
** Bold : contient "bold", "heavy", "black"
** Italic : contient "italic", "oblique", "slanted"
** Monospace : courier, mono, consolas, menlo, jetbrains, fira code, source code, dejavu sans mono, liberation mono
** Bold+Italic : "BoldItalic" → BOLD_ITALIC

=== 3. Hiérarchie multi-niveaux

* Ancienne logique : 2 niveaux (chapter ≥ threshold, section ≥ threshold) basée sur avg + range
* Nouvelle logique : 3 niveaux (h1/h2/h3) basée sur `max * ratio`
** h1 ≥ `max * 0.95` (et bold pour = , sinon ==)
** h2 ≥ `max * 0.75` (et bold pour ==, sinon ===)
** h3 ≥ `max * 0.50` (et bold pour ===, sinon ignoré)
* `writeString` remplace `processTextPosition` → texte reconstruit proprement, pas de fragments caractère par caractère
* Grouping simplifié : plus de Y-grouping complexe, `writeString` donne déjà des lignes

=== 4. Blocs de code monospace

* Détection : si `fontStyle.isMonospace()` → accumulate lines
* 2+ lignes monospace consécutives → `[source,text]` + `----`
* 1 ligne monospace seule → `backtick` inline

=== 5. Tests unitaires

* `ExtractBookStructureTaskTest.kt` (288 lignes, 10 tests)
* PDFBox création de PDF synthétiques via `PDDocument`/`PDPageContentStream`
** 4 tests `FontStyleDetector` (bold, italic, monospace, normal, bolditalic)
** 3 tests `extractBookStructure` (headers + code, vide, code-only)
** 1 test `hierarchy` (hiérarchie multi-niveaux)
** 1 test `inline code` (single line monospace)
** 1 test `empty PDF`
* Résultat : **10/10 PASS**
